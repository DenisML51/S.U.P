# Иконки приложения

## Текущая иконка

Сейчас используется базовая иконка `icon.svg`. 

## Замена иконки для Windows

1. **Создайте иконку** размером 256x256 или 512x512 пикселей в формате PNG
2. **Конвертируйте в ICO** используя один из способов:

### Онлайн конвертеры:
- https://convertio.co/png-ico/
- https://cloudconvert.com/png-to-ico
- https://iconverticons.com/online/

### Инструменты:
- **ImageMagick**: `magick convert icon.png -define icon:auto-resize=256,128,64,48,32,16 icon.ico`
- **GIMP**: Экспорт как ICO
- **Photoshop**: Сохранение для Web как ICO

3. **Поместите файл** `icon.ico` в папку `assets/`

4. **Пересоберите приложение**:
   ```bash
   npm run build:win
   ```

## Рекомендации по иконке

- **Размер**: 256x256 или 512x512 пикселей
- **Формат**: PNG с прозрачным фоном (для лучшего качества)
- **Стиль**: Простая, узнаваемая, хорошо читается в маленьком размере
- **Цвета**: Соответствуют дизайну приложения (синий/фиолетовый градиент)
